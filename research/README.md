# Research Papers

This directory contains research papers and technical documents in markdown format.

## Adding New Content

To add a new research paper:
1. Create a new .md file in this directory
2. Name it descriptively (e.g., `zero-day-exploitation-techniques.md`)
3. Add your content in markdown format

## Accessing Content

Your content will be accessible at:
- `https://oppenheim3r.github.io/research/your-file-name.md`
