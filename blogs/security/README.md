# Security Blog Posts

This directory contains security-related blog posts in markdown format.

## Adding New Content

To add a new security blog post:
1. Create a new .md file in this directory
2. Name it descriptively (e.g., `vulnerability-analysis-methodology.md`)
3. Add your content in markdown format

## Accessing Content

Your content will be accessible at:
- `https://oppenheim3r.github.io/blogs/security/your-file-name.md`
