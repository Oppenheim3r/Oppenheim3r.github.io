# Projects

This directory contains project documentation in markdown format.

## Adding New Content

To add a new project:
1. Create a new .md file in this directory
2. Name it descriptively (e.g., `network-security-tool.md`)
3. Add your content in markdown format

## Accessing Content

Your content will be accessible at:
- `https://oppenheim3r.github.io/projects/your-file-name.md`
